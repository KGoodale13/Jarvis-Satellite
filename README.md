# Jarvis-Satellite
A HomeAssistant Wyoming Satellite. Runs on RPI4 with a DigiAmp+ and Respeaker XMOS3800 mic array


On a fresh Raspberry pi 4+ running Raspberry Pi OS (64 bit) lite run the following:

```bash
wget -qO- https://raw.githubusercontent.com/KGoodale13/Jarvis-Satellite/refs/heads/main/install_hook.sh | bash
```
